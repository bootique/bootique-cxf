# bootique-cxf

Provides [Apache CXF](https://cxf.apache.org/) integration with [Bootique](http://bootique.io).

CXF can be used in a different contexts:
- [JAX-WS Server](bootique-cxf-jaxws-server)
- [JAX-RS Server](bootique-cxf-jaxrs)
